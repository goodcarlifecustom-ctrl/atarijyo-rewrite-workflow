# 装飾プロンプト

リライト後の記事を、WordPressテーマ「SWELL」で読みやすくなるように装飾してください。

## 基本ルール

- 読みやすい位置で改行する
- 重要な結論には太字またはマーカーを使う
- 箇条書きにできる箇所はul/olで整理する
- 注意点はボックス化する
- 比較できる内容は必要に応じて表にする
- 冒頭に読者の悩みに寄り添う吹き出しを入れる
- 内部リンクは自然なアンカーテキストで挿入する
- 記事内ジャンプリンクが有効な箇所にはアンカーを設置する

## 装飾対象

- 導入文
- 比較・選び方
- 注意点
- メリット・デメリット
- 手順
- まとめ
- アフィリエイト導線の直前
- 内部リンク周辺

## 保持するもの

- 既存のSWELLブロック
- ショートコード
- アフィリエイトリンク
- 広告タグ
- 計測タグ
- 既存の比較表
- 既存の画像

## 禁止事項

- アフィリエイトリンクを勝手に削除しない
- ショートコードを勝手に変更しない
- SWELLブロックコメントを壊さない
- 競合サイトの見出しをそのままコピーしない
- KWを不自然に詰め込まない
- 装飾を過剰に入れすぎない
