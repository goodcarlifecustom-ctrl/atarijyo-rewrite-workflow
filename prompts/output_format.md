# 出力形式

1記事の処理が完了したら、以下のファイルを出力してください。

## output/rewrite_report.md

以下を含めてください。

- 対象記事URL
- メインKW
- Search Console分析結果
- Ahrefs分析結果
- 追加候補KW
- KW分類
- 追加した見出し
- 追加した本文要素
- 内部リンク追加案
- 公的リンク追加案
- アフィリエイト導線の改善案
- 手動確認が必要な箇所

## output/rewrite_review.html

確認用HTMLです。
追加・修正箇所に色付けしてください。

例：

```html
<span class="rewrite-added">追加箇所</span>
<span class="rewrite-modified">修正箇所</span>
```

## output/rewritten.html

WordPress下書き投入用HTMLです。
色付けタグは削除してください。

## output/delete_suggestions.md

削除候補を理由付きで提案してください。
ただし、自動削除はしないでください。
